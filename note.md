3 stages 
```
U =Untracked
A = Added
M = Modified
```

git initialization
```
git init
```

add file
```
git add FILENAME
```
add all files & folders
```
git add .
```

commit
```
git commit -m "your message"
```

push 1st time
```
git push origin main
```

push 2nd to last time
```
git push
```

git clone
```
git clone "repository link"
```
